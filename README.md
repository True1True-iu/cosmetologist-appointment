# Cosmetologist Appointment

Приложение для записи к косметологу.

## Структура проекта

- `frontend` — клиентская часть на React + Vite.
- `backend` — серверная часть на Node.js + Express.

## Быстрый старт

1. Установите зависимости:

```bash
npm install
npm install --prefix backend
npm install --prefix frontend
```

2. Запустите проект (frontend и backend вместе):

```bash
npm run dev
```

## Полезные команды

- `npm run dev` — запуск frontend и backend одновременно.
- `npm run dev:backend` — запуск только backend.
- `npm run dev:frontend` — запуск только frontend.
- `npm run build:frontend` — сборка frontend.
